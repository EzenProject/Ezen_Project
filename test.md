## Git CommandLine
- 파일 추가
  - git add . ( .은 전체를 의미한다. 즉  폴더내 전체 파일을 업로드 )
- 파일 커밋( 내용 추가 )
  - git commit -m "커밋할 내용"
- 업로드
  - git push <원격 저장소> <지역 브랜치>
  - git push origin master
